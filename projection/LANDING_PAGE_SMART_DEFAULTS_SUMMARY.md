### Key Calculations

**Portfolio at Retirement:**
```
FV = PV * (1 + r)^n + PMT * [((1 + r)^n - 1) / r]

Where:
- PV = Current balance
- r = Annual return rate
- n = Years to retirement
- PMT = Annual contribution
```

**Monthly Income (4% Rule):**
```
Monthly = (Portfolio @ Retirement * 0.04) / 12
```

**Confidence Level:**
```
If Portfolio/Starting > 15x ‚Üí Comfortable (85%)
If Portfolio/Starting > 8x  ‚Üí Borderline (65%)
If Portfolio/Starting > 4x  ‚Üí Borderline (45%)
Otherwise                   ‚Üí Low (30%)
```

**Q: Can I customize the default values?**
A: Yes! Edit the `DEFAULT_VALUES` object in `defaultValues.ts` to customize any strategy.
